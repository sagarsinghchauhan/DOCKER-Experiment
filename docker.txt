install Docker Desktop 
sign in to Dockerhub and Docker Desktop
Docker Desktop:
    > check docker install: cli -> "docker"
    > Pull the hello-world image and run it 

Build docker image: docker build -t flask-docker-demo
Run the container: docker run -p 5000:5000 flask-docker-demo
Tag your image: docker tag flask-docker-demo sagar/flask-docker-demo:lastest1
Assure that you're logged in: docker login 
Push image to dockerhub:docker push sagar/flask-docker-demo:lastest1
Pull image from dockerhub: docker pull sagar/flask-docker-demo:lastest1
Run the pulled image: docker run -p 5000:5000 sagar/flask-docker-demo:lastest1
